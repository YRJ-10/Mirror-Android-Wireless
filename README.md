# Wireless Mirror

Wireless Mirror adalah launcher sederhana untuk menjalankan mirror layar hp android (scrcpy) secara wireless di Windows.

![Tampilan aplikasi](sampel.png)

## Cara Pakai dari Source

Clone repository:

```bash
git clone <url repo>
cd Mirror-Android-Wireless
```

Install dependency:

```bash
npm install
```

Jalankan saat development:

```bash
npm start
```

Build aplikasi Windows:

```bash
npm run build
```

Hasil build ada di folder:

```text
dist/Mirror Wireless-win32-x64
```
